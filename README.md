# [1번 문제] 함수의 활용문제
☘️ **문제**

<br/>

<br/>

**test case**

| num | total | result |
| --- | --- | --- |
| 3  | 12 | [3,4,5] |
| 4 | 14 | [2,3,4,5] |
| 5 | 5 | [-1,0,1,2,3] |

**힌트**

- `평균값`을 활용해보세요!
- 배열의 `첫번째 요소`의 **패턴을 찾아 공식화** 해보세요!

# [3번 문제] 객체의 활용
☘️ **문제**

주어진 likelion 객체 배열을 다음 조건에 맞게 활용해주세요!

1. **나이순으로(오름차순)으로** 정렬해주세요!
2. **24세 이상** 팀원만 추출하여 필터링해주세요!

**주의**

- **`map`** 이외의 `sort` , `filter` 등과 같은 메서드는 추후세션의 이해를 위해 사용을 지양해주세요!
