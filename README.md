# [1번 문제] 함수의 활용문제
☘️ **문제**


함수의 매개변수로 `num`과 `total`이 주어집니다. 연속된 수 num개를 더한 값이 total이 될 때, 이 정수 배열을 오름차순으로 담아 return하도록 solution함수를 완성해보세요!
예를들어 `num`이 3이고 `total`이 12인 경우 배열은 [3,4,5]입니다. <br/><br/>

<br/>

**test case**

| num | total | result |
| --- | --- | --- |
| 3  | 12 | [3,4,5] |
| 4 | 14 | [2,3,4,5] |
| 5 | 5 | [-1,0,1,2,3] |

**힌트**

- `평균값`을 활용해보세요!
- 배열의 `첫번째 요소`를 **패턴을 찾아 공식화** 해보세요!
